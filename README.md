# Ultrakill Custom Style Names

A fun mod that replaces standard style action names with custom ones. Experience ULTRAKILL with a bit more flair!

> **Note:** This is an Alpha version, so not all style bonuses have been renamed yet. Expect updates in the future. Everything else is currently under development.

## Features
- Replaces **Arsenal** with **ADHD**
- Replaces **Ricoshot** with **Tax Evasion**
- Replaces **Kill** with **APPETIZER**
- Replaces **Big Kill** with **FAT L**
- Replaces **Fireworks** with **GENDER REVEAL**
- Replaces **Headshot Combo** with **SPEEDRUN GENOCIDE**
- Replaces **Disrespect** with **Ew**
- Replaces **Parry** with **Nuh uh!**
- Replaces **Projectile Boost** with **THINK FAST CHUCKLENUTS**
- Replaces **Fistful of Dollar** with **McDuck**
- Replaces **Doublekill** with **STONKS**
- Replaces **Triplekill** with **GASLIGHT GATEKEEP GIRLBOSS**
- Replaces **Multikill** with **ULTRA-VIOLENCE**
- Replaces **Fist Kill** with **Mid**
- Replaces **Big Fist Kill** with **Git gud**
- Replaces **Headshot** with **LOBOTOMY**
- Replaces **Big Headshot** with **BRAINFUCK**
- Replaces **Enraged** with **cry bout it**

## Installation
1. Install **BepInEx**.
2. Unzip **BepInEx** to the **same folder** as **ULTRAKILL.exe**.
3. Drag `Plugin.dll` from **UKStyleEditor/plugins** into your `ULTRAKILL/BepInEx/plugins` folder and launch the game.

## Requirements
- BepInEx 5.4.21 or higher

## Compatibility
- Works on both **Licensed** and **Pirated** versions of the game.
